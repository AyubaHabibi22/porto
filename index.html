Tentang Saya

Yuk, belajar web programming

Halo! Saya Ayuba Habibi, seorang mahasiswa Teknik Informatika yang memiliki minat besar dalam pengembangan perangkat lunak dan teknologi informasi. Saya sedang belajar membangun aplikasi web menggunakan berbagai teknologi, termasuk HTML, CSS, JavaScript, dan framework seperti Laravel. Saya adalah pribadi yang detail, suka memecahkan masalah, dan selalu bersemangat untuk belajar hal-hal baru. Website portofolio ini berisi berbagai proyek yang telah saya kerjakan sebagai bukti dari kemampuan dan dedikasi saya di bidang teknologi. Tujuan saya adalah terus berkembang sebagai developer profesional dan berkontribusi dalam menciptakan solusi digital yang bermanfaat.

Mari Berteman

Saya senang berkenalan dan terhubung dengan orang-orang yang memiliki minat yang sama di dunia teknologi. Jika kamu ingin berdiskusi tentang pemrograman, kolaborasi proyek, atau sekadar berbagi pengalaman, jangan ragu untuk menghubungi saya melalui media sosial di bawah ini. Mari kita tumbuh dan belajar bersama sebagai sesama pengembang!.

Project Terbaru

Landing Page Ayuba Habibi

Sebuah landing page pribadi yang menampilkan profil saya sebagai Gamer dan Analyst. Dibuat dengan HTML, CSS, dan JavaScript. Halaman ini dirancang untuk memperkenalkan diri secara profesional, lengkap dengan deskripsi, tombol call-to-action, dan navigasi yang responsif.

E-Commerce
